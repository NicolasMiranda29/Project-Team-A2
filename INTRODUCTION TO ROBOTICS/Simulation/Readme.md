# PPP Robot Simulation in MATLAB

## üìå Project Overview
In this project, a **PPP-type robot** with a **port crane-style structure** was implemented using MATLAB.  
The project demonstrates kinematics, dynamics, visualization, and real-time control through a graphical interface.

---

## üìÇ Folder Structure
The folder **`PPP_Robot_GUI`** contains the main functions:

- **`ppp_fk.m`** ‚Üí Direct kinematics model  
- **`plot_ppp.m`** ‚Üí Robot visualization  
- **`animate_pick_and_place.m`** ‚Üí Example of a joint-space trajectory  
- **`ppp_dynamics.m`** ‚Üí Simplified dynamic model  
- **`gui_ppp_sim.m`** ‚Üí Graphical interface with sliders for real-time control  

---

## üöÄ Features
- Direct kinematics computation for PPP-type robot  
- Real-time visualization of robot motion  
- Example trajectory for pick-and-place tasks  
- Simplified dynamic modeling  
- Interactive GUI with sliders for intuitive control  

---

## ‚ñ∂Ô∏è Usage
1. Clone or download this repository.  
2. Open MATLAB and navigate to the `PPP_Robot_GUI` folder.  
3. Run `gui_ppp_sim.m` to launch the graphical interface.  
4. Explore the other scripts for kinematics, dynamics, and trajectory examples.
